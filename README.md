# JW 純字幕擷取器 (JW Subtitle Extractor)

一個專為閱讀、做筆記與語言學習設計的輕量網頁工具。使用者只需貼上 JW 官方影片的分享網址，程式便會全自動解析 API，並透過**「高效逐行過濾機制」**，將原本繁雜的字幕檔案提煉為乾淨、流暢的純文字文章。

## 🌟 核心特色
- **北歐現代淺色調 (Nordic Light Theme)**：採用清爽、明亮的明色系介面，與播放器專案做出完美視覺區隔，長時間閱讀不易疲勞。
- **極致文字清洗機**：
  - 自動蒸發 `WEBVTT` 標頭、區塊 ID 與所有時間軸。
  - 獨家採用**逐行審查法（Looping Lines）**，完美保留字幕本身的換行，同時徹底拔除行與行之間多餘的空白行。
- **動態標題偵測**：擷取字幕的同時，全自動撈取並顯示影片的真實中文標題。
- **一鍵複製與動態反饋**：採用現代 `navigator.clipboard` API 實現流暢的一鍵複製，並內建貼心的按鈕顏色動態反饋。

## 🛠️ 技術棧 (Tech Stack)
- HTML5 / CSS3 (響應式淺色字型排版、`pre` 滾動條優化)
- JavaScript (ES6+, Async/Await、陣列逐行過濾、正則表達式、剪貼簿 API)

## 🚀 如何使用
1. 將本專案下載或複製（Clone）到本機電腦。
2. 點擊 `index.html` 在瀏覽器中打開。
3. 貼上 JW 網站的影片分享連結。
4. 點擊「擷取字幕」按鈕，即可瞬間獲得排版完美的純文字字幕，點擊複製即可黏貼至個人筆記（如 Notion、Obsidian）。
